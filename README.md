# 🚀 Space Invaders

A classic arcade-style space shooter game built with Python and Pygame. Defend your base from waves of invading aliens!

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![Pygame](https://img.shields.io/badge/Pygame-2.0+-green.svg)
![License](https://img.shields.io/badge/License-MIT-yellow.svg)

## ✨ Features

- 🎮 **Classic Gameplay**: Authentic Space Invaders experience with modern enhancements
- 🎨 **Retro Graphics**: Pixel-perfect sprites and immersive background
- 🔊 **Sound Effects**: Laser blasts and explosion sounds for maximum immersion
- ⚡ **Optimized Performance**: 
  - Smooth 60 FPS gameplay
  - Preloaded audio assets for zero-latency sound
  - Efficient rendering pipeline
- 🎯 **Progressive Difficulty**: Enemies get faster as you advance

## 📋 Requirements

- Python 3.8 or higher
- Pygame 2.0 or higher

## 🚀 Installation

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd <project-directory>
   ```

2. **Install dependencies:**
   ```bash
   pip install pygame
   ```

## 🎮 How to Play

1. **Start the game:**
   ```bash
   python main.py
   ```

2. **Controls:**
   - **← → Arrow Keys**: Move left/right
   - **Spacebar**: Fire laser
   - **ESC**: Quit game

3. **Objective:**
   - Destroy all alien invaders before they reach your base
   - Avoid enemy fire
   - Achieve the highest score possible!
